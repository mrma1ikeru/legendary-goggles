# Telegram Recipe Bot 🍳

Telegram-бот для поиска рецептов из продуктов, которые есть в холодильнике.

## Возможности
- поиск рецептов;
- выбор языка;
- перевод рецептов;
- избранное;
- фотографии блюд.

## Стек технологий
- Python
- Aiogram
- TheMealDB API
